# Sales Super-Skill

A comprehensive end-to-end sales skill merging **Perplexity Computer's 6 built-in sales skills** with **Claude Code's customer interview analysis, brand voice analysis, content research, RICE prioritization, and UX research**.

## What's Inside

| Section | Description |
|---------|-------------|
| Gap Analysis | Full capability comparison matrix |
| Account Research & Prospect Intelligence | Company/person research + customer interview analysis + persona development |
| Call Prep & Meeting Readiness | Meeting playbooks, SPIN discovery, attendee research, objection handling |
| Competitive Intelligence & Positioning | Competitor research, battlecards, brand voice positioning, win/loss analysis |
| Outreach & Content Creation | Research-first AIDA outreach, LinkedIn messaging, follow-up sequences |
| Sales Assets & Deliverables | Landing pages, decks, one-pagers, workflow demos |
| Deal Prioritization & Pipeline | Daily briefings, RICE-adapted deal scoring, pipeline management |
| Unique Perplexity Capabilities | Live web research, AI generation, 400+ integrations, scheduled monitoring |
